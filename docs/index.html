Ambil CLI

CLI Application to download default config file

Install

npm install -g ambil-cli

Help

ambil --help

ambil <cmd> [args]

Commands:
  ambil get [name]  Get config file
  ambil list        Show all config list

Options:
  --version  Show version number  [boolean]
  --help     Show help            [boolean]

Basic Usage

Show all configs

ambil list
